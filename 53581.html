U盘向：启动盘恢复为普通U盘

发表于 2019-03-07 更新于 2023-01-17 分类于 实用教程 阅读次数：
本文字数： 357 阅读时长 ≈ 1 分钟

介绍

  • 在重装系统时，我们会将U盘做成系统启动盘，本篇教程就是如何将启动盘恢复成普通U盘。
  • 如果U盘内含有重要资料，请提前备份到电脑上，因为下面的步骤将会格式化你的U盘。

方法

cmd命令行方法

  • 此电脑右击-管理-磁盘管理，在这里先确定好你的U盘是在哪个磁盘上，例如：我这里是在磁盘2
  • 使用Win+R组合键，输入 cmd 并回车
  • 输入 diskpart 并回车
  • 在新弹出的页面输入 select disk 2 并回车（这里的2就是指的磁盘2，请根据自己电脑自行修改数字）
  • 输入 clean 并回车，直到提示你清除完毕
  • 在第一步的磁盘管理中找到你的U盘，右键-新建简单卷-一直下一步-直到新建完成
  • 这时候你的U盘就恢复为普通U盘了

制做启动盘工具恢复法

  • 找到你当初制作启动盘的软件，部分软件含有恢复功能，直接傻瓜式操作即可
  • 如果你制盘工具没有该功能，请参照上面的方法
